# APDL (APDS Protocol Definition Language)

APDL（APDS协议定义语言）是一个面向航天领域的协议定义与仿真验证一体化平台，旨在提供CCSDS标准协议的灵活组合、全流程软件仿真和性能分析能力。

## 项目概述

APDL系统目标：
- 提供CCSDS协议的标准化定义工具
- 实现纯软件协议仿真验证平台
- 构建协议性能分析引擎
- 支持协议规范自动生成

## 核心功能

### 协议定义与扩展模块
- 预实现CCSDS全系列标准协议的语法单元
- 支持TM（遥测）、TC（遥控）、AOS（高级在轨系统）等标准协议层
- 提供轻量级声明式DSL，专为CCSDS协议组合设计
- 支持语法单元的热插拔扩展

### 文档解析与预处理模块
- 支持Word、Excel、PDF、表格等多种文档格式
- 支持宽松输入适配
- 基于规则引擎+浅度NLP提取核心信息

### 协议规范生成模块
- 内置航天领域协议规范标准模板
- 支持多格式导出（Word、PDF、Excel、Markdown）

### 纯软件仿真内核模块
- 数据模拟生成
- 打包/复接引擎
- 传输链路模拟
- 接收/解复接/拆包引擎

### 协议检验与性能分析模块
- 协议合理性检验
- 性能统计分析
- 可视化分析报告

## 技术栈

- **编程语言**: Rust
- **架构**: 模块化解耦设计
- **协议标准**: CCSDS（空间数据系统咨询委员会）标准

## 核心设计理念

1. **平台-语法单元分离**: 通过Rust Trait实现协议单元的热插拔扩展
2. **DSL描述语言**: 支持复杂协议栈的灵活定义
3. **纯软件仿真**: 无需硬件依赖的端到端协议仿真
4. **模块化架构**: 严格解耦的微模块设计

## 使用场景

- 航天链路协议开发
- CCSDS协议组合验证
- 协议性能分析
- 仿真验证平台
- 协议规范自动生成
